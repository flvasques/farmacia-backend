<p align="center">API em Laravel 12</p>

## Sobre
Uma API rest utlizando Laravel 12

## Pré requisitos:
- PHP 8.2 ou superior;
- Composer;
- Mysql;

## Instalação Local

- 1 Criar o arquivo .env apartir do arquivo .env.example;
- 2 Sustituir as credencias do banco de dados pelas do banco de dados local;
- 3 Executar
 ```bash
  composer install
  php artisan migrate --seed
 ```

## Execução

- Executar
```bash
 php artisan serve
```
